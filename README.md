Threads Master 🧵✨Threads Master — это легковесное, работающее прямо в браузере AI-приложение, объединяющее функционал генератора вирусных "хуков" (зацепок) и умного комментатора для соцсети Threads.Инструмент помогает авторам быстрее создавать цепляющий контент и генерировать органический трафик за счёт продуманных, нативных комментариев к чужим постам.🚀 ВозможностиПриложение состоит из двух основных модулей:Генератор ХуковВыдаёт сразу 10 вариантов начала для вашего поста.Использует 5 проверенных копирайтерских фреймворков: Слом шаблона, Личная история, Агрессивный старт, Инсайд и Поляризация.Умный КомментаторГенерирует 3 варианта ответа на любой чужой пост.Поддерживает выбор тональности: Дружелюбный, Экспертный, Ироничный, Вопрос.ИИ настроен избегать "роботизированных" фраз, хештегов и пишет максимально естественно, подстраиваясь под язык оригинала.🛠 Установка и запускПроект состоит всего из одного файла index.html. Никаких сборок (npm/webpack) не требуется!Склонируйте репозиторий или скачайте файл index.html.Откройте файл index.html в любом текстовом редакторе.Откройте приложение в браузере и вставьте ваш Gemini API-ключ в верхнее поле «Вставьте ваш Gemini API ключ сюда...». Ключ сохраняется локально в localStorage браузера и не зашивается в код.Просто откройте файл index.html в любом браузере двойным кликом (или загрузите на GitHub Pages / Vercel для публичного доступа).💻 Стек технологийHTML5 / Vanilla JavaScriptTailwind CSS (через CDN для стилизации)FontAwesome (иконки)Google Gemini API (в качестве языковой модели). В коде используется безопасное хранение ключа в браузере и fallback по моделям для снижения риска 404.🎨 Скриншоты / ДизайнДизайн выполнен в темных тонах (Dark Mode), имитирующих минималистичный интерфейс оригинального приложения Threads. Все элементы адаптивны и прекрасно работают как на ПК, так и на смартфонах.Создано с помощью Claude / Gemini AI.
